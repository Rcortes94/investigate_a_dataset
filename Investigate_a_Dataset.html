#defining function to split the genre values in a specific character
def count_genre(gen):
    genre = df[gen].str.cat(sep='|')
    genre= pd.Series(genre.split('|'))
    count = genre.value_counts(ascending =False)
    return count

Most Profitable Movie¶

To analyze the most profitable movie I used the function max_min that return the maximum value of the column specified.

Function max_min:¶

The function max_min that in a column as a parameter and it uses the built-in panda function idxmax() that returns the index of the maximum value in the column. Assign the index to the variable max_v and it is later use to find the row with the corresponding index. Same process is repeated for the min value but with idxmin() instead. After both rown have been extracted we use de concat function to stich them together and built a temporary dataframe with information we need.

In [59]:
#defining the function max_min that get the max and minimus value from the column specified

def max_min(column):
    # Max profit
    max_v = df[column].idxmax()
    max_v = df.loc[max_v]
   
    # Min profit
    min_v = df[column].idxmin()
    min_v = df.loc[min_v]
    
    #Concat the to columns to make a single dataframe
    df_info = pd.concat([max_v, min_v], axis=1)
    
    return df_info
